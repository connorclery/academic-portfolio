# Academic Portfolio Website

A modern, professional academic portfolio website template designed to showcase your academic profile, research, education, and publications.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Layout**: Clean and modern design suitable for academic profiles
- **Easy to Customize**: Simple HTML, CSS, and JavaScript structure
- **Smooth Animations**: Elegant transitions and scroll effects
- **Sticky Navigation**: Quick access to different sections
- **Timeline for Education**: Visual representation of your educational background
- **Research Interests**: Highlight your areas of focus
- **Publications List**: Display your academic publications
- **Contact Section**: Easy ways for visitors to reach you

## Sections

1. **Navigation Bar**: Quick links to all sections
2. **Hero Section**: Your name, title, and profile picture
3. **About Me**: Brief biography and background
4. **Education**: Timeline of your educational achievements
5. **Research Interests**: Your main areas of research focus
6. **Publications**: List of your academic publications
7. **Contact**: Links to email, LinkedIn, GitHub, etc.

## Getting Started

### Prerequisites
- A web browser
- A text editor (VS Code, Sublime Text, etc.)
- Git (optional, for version control)

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Edit the content to match your information

## Customization

### Update Your Information

Edit `index.html` to:
- Replace "Your Name" with your actual name
- Update your academic title and research focus
- Add your profile picture URL
- Fill in your education details
- Update your research interests
- Add your publications
- Update contact information

### Customize Colors

Edit `styles.css` to change the color scheme:

```css
:root {
    --primary-color: #2c3e50;      /* Dark blue */
    --secondary-color: #3498db;    /* Light blue */
    --text-color: #333;             /* Dark gray */
    --light-bg: #ecf0f1;            /* Light gray */
}
```

### Add More Sections

You can easily add new sections by following the existing pattern:

```html
<section id="new-section" class="section">
    <div class="container">
        <h2>Section Title</h2>
        <!-- Your content here -->
    </div>
</section>
```

## Deployment

### GitHub Pages
1. Push your repository to GitHub
2. Go to repository settings
3. Enable GitHub Pages with the `main` branch
4. Your site will be live at `https://yourusername.github.io/academic-portfolio`
